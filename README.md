## Installation
1. `git clone https://github.com/DewDewTeam/laravel-ecommerce.git`

2. `cd laravel-ecommerce`

3. `composer install`

4. config database in **.env** file

5. `php artisan migrate`

6. `php artisan serve`

**Starting Laravel development server: http://127.0.0.1:8000**
